Emacs Setup up!

Press Down Arrow!!!

Install ox-revel

(use-package ox-reveal
  :ensure t
  :config
  (progn
    (setq org-reveal-root "http://cdn.jsdelivr.net/reveal.js/3.0.0/")
    (setq org-reveal-mathjax t)))

Install htmlize

(use-package htmlize
  :ensure t)
